Продається Mystery MID-703G

Новий Mystery MID-703G підкреслить яскраву індивідуальність свого власника, подарує свіжі емоції й можливість відчути світ інтерактивного контенту по-новому.

Потужна апаратна складова Двоядерний процесор ідеально підходить для багатозадачності. Ефективні компоненти дадуть змогу працювати з ресурсомісткими програмами, забезпечують безперебійний перегляд відео та прекрасно підходять для ігор. Процесор Mediatek з тактовою частотою 1.3 ГГц разом із графічним співпроцесором Mali-400 MP роблять цей планшет зручним і багатоцільовим інструментом для роботи та розваг.

Екран 7" (800х480) ємнісний, MultiTouch / Mediatek MT8312 (1.3 ГГц) / RAM 512 МБ / 4 ГБ вбудованої пам'яті + microSD / Wi-Fi / Bluetooth / основна камера 2 Мп, фронтальна 0.3 Мп / 3G / GPS / Android 4.2 (Jelly Bean)
